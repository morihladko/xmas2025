# Virtual Christmas Postcard 2025

A festive 3D Christmas postcard featuring a snowy cabin scene.

**[See it in action](https://morihladko.github.io/xmas2025)**

## Built With

- [Claude Code](https://claude.ai/claude-code) - AI-powered coding assistant
- [React](https://react.dev/) - UI framework
- [Vite](https://vite.dev/) - Build tool
- [React Three Fiber](https://r3f.docs.pmnd.rs/) - WebGL/Three.js for React
- [Tripo AI](https://www.tripo3d.ai/) - 3D model generation
